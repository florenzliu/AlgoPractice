# Array

## Time Complexity Analysis

| Type | Insert | Delete | Update | Search | Traverse |
| :--- | :--- | :--- | :--- | :--- | :--- |
| Array | O\(n\) | O\(n\) | O\(1\) | O\(1\) | O\(n\) |

## Operations

* each\(\): traverse
* pop\(index\): delete element by its index
* insert\(item, index\): insert item at index
